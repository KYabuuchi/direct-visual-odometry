## Description
* [semi-dense-visual-odometry](https://jsturm.de/publications/data/engel2013iccv.pdf)の再現実装．
* 2019年東京大学五月祭での展示用に作成．
* ただし回転の追従がうまくいかず，展示時には回転運動が生じないように拘束して運用した
    * for-exhibitionブランチはそのときのそれ
